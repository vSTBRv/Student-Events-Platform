# Uruchomienie frontendu
* Aby uruchomić aplikację frontendową, wykonaj poniższe kroki:

# Przejdź do katalogu projektu:
* cd frontend

#  Zainstaluj zależności:
* npm install

# Uruchom środowisko deweloperskie:
* npm run dev

# Po uruchomieniu aplikacja będzie dostępna pod adresem wyświetlonym w konsoli (zazwyczaj http://localhost:5173
